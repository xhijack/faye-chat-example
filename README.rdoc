== Faye Example Chat

Faye Example Chat is an example how to use faye and rails.

== Getting Started

1. <tt>git clone https://github.com/xhijack/faye-chat-example</tt>

2. ./faye.sh
    this is for running faye

3. <tt>rails s</tt>

4. run localhost:3000 on your browser
